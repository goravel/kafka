# Kafka

A Kafka driver for `facades.Queue()` of Goravel, powered by [franz-go](https://github.com/twmb/franz-go).

## Version

| goravel/kafka | goravel/framework |
| ------------- | ----------------- |
| v1.18.x       | v1.18.x           |

## Install

Run the command below in your project to install the package automatically:

```
./artisan package:install github.com/goravel/kafka
```

Or check [the setup file](./setup/setup.go) to install the package manually.

## Testing

Run command below to run test:

```
go test ./...
```
